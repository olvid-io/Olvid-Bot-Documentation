Nous avons développé deux noeuds communautaires distincts pour interagir avec l'ecosystème Olvid depuis n8n. 


:::{card}
:width: 100%
:link: olvid
:link-type: doc
**💬 Olvid**
^^^
Découvrez notre noeud *Olvid*  pour piloter un daemon dans n8n. 
Il vous permet à la fois de réaliser rapidement des flows d'alertes ou des Chat Bots, et de créer des intégrations bien plus complexes.
:::

:::{card}
:width: 100%
:link: keycloak
:link-type: doc
**🔑 Keycloak**
^^^^
Découvrez notre noeud *OlvidKeycloak*.
Pour les utilisateurs d'un annuaire annuaire Olvid dans Keycloak, il vous permet de gérer votre annuaire comme dans la console d'administration. 
:::
